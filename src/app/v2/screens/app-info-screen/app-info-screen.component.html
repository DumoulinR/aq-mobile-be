Hoe werkt dit?

We beoordelen de impact van de algemene luchtkwaliteit op korte termijn op basis van een luchtkwaliteitsindex.

Die geeft aan de meetwaarden van verschillende verontreinigende stoffen één score (Heel slecht). De index baseert zich voor deze score op data van meetstations in heel België.

De stoffen die we momenteel in rekening brengen, zijn:

  • Ozon (O3)

    78 µg/m3

  • Ozon (O3)

    78 µg/m3

  • Ozon (O3)

    78 µg/m3

  • Ozon (O3)

    78 µg/m3

Door op deze data modellen toe te passen, berekenen we of de impact van de luchtkwaliteit op lange termijn beter of slechter wordt.

Je kan deze inschattingen bekijken in het menu via ‘blootstelling op lange termijn’.

Hoe worden deze waarden berekend?

Als de metingen de waarschuwings- waarde overschrijden, sturen we je een melding. Je kan je meldingen beheren via het menu.
